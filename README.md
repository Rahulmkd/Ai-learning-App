# AI Learning App 📘🤖

An AI-powered learning application that helps users **learn, summarize notes, and extract insights from PDFs** using modern AI technologies.

---

## 🚀 Features

- Upload and summarize PDFs
- Learn from personal notes
- AI-generated summaries
- Secure authentication (JWT)
- Clean UI with React + Tailwind
- RESTful API (Node.js & Express)

---

## 🛠️ Tech Stack

**Frontend**

- React
- Tailwind CSS
- Vite

**Backend**

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer (PDF upload)
- Google GenAI
